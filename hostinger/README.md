## 🚀 Quick Start (3 Steps)

### Step 1: Upload to Hostinger
1. Download this entire `hostinger-deploy/` folder
2. Upload all contents to `public_html/` via File Manager or FTP
3. Make sure `.htaccess` is in the root of `public_html/`

### Step 2: Create Database
1. Go to Hostinger hPanel → Databases → MySQL Databases
2. Create database (e.g., `u123456789_artistiya`)
3. Note: DB Name, DB User, DB Password

### Step 3: Run Auto-Installer
1. Visit `https://yourdomain.com/install.php`
2. Enter database credentials → schema installs automatically
3. Create admin account → installer self-deletes
4. Done! Your API is live at `/api/*`

## 🔑 After Installation

### Configure Services (Admin Dashboard → Settings)
- **Email**: SMTP credentials (Hostinger: smtp.hostinger.com:465)
- **SMS**: Gateway API keys
- **Payments**: bKash, SSLCommerz, etc.
- **Delivery**: Pathao, Steadfast, etc.

### Frontend Connection
Update `src/lib/api.ts` in your React app:
```typescript
const API_BASE = 'https://yourdomain.com/api';
```

## 🔒 Security Notes
- `.env.php` is auto-generated and protected by `.htaccess`
- `install.php` auto-deletes after setup
- All passwords use Argon2ID hashing
- JWT tokens for API authentication
- AES-256 encryption for stored credentials
